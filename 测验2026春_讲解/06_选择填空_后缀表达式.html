🧮 后缀表达式求值 — 栈的9步变化

后缀表达式：5 2 3 1 + * 2 / +
求计算结果 = ?
正确答案
结果 = 9
计算过程：5 2 3 1 + * 2 / + = 5 2 4 * 2 / + = 5 8 2 / + = 5 4 + = 9

一、后缀表达式求值规则

后缀表达式（逆波兰式）的求值只需要一个栈：

1
遇到操作数（数字）→ 压栈（push）
2
遇到操作符（+ - * /）→ 弹出两个操作数，计算后将结果压栈
注意顺序：先弹出的是右操作数，后弹出的是左操作数。这对减法和除法很重要！
3
表达式扫描完后，栈中仅剩一个元素 → 最终结果

二、9步栈状态完整追踪

步骤 1-3：连续压入三个操作数

═══ 步骤1：扫描到 5 ═══ 操作：push 5 栈状态： ┌───┐ │ 5 │ ← 栈顶（也是栈底） └───┘ 栈 = [5] ═══ 步骤2：扫描到 2 ═══ 操作：push 2 栈状态： ┌───┐ │ 2 │ ← 栈顶 │ 5 │ └───┘ 栈 = [5, 2] ═══ 步骤3：扫描到 3 ═══ 操作：push 3 栈状态： ┌───┐ │ 3 │ ← 栈顶 │ 2 │ │ 5 │ └───┘ 栈 = [5, 2, 3]

步骤4：扫描到 1 → push

═══ 步骤4：扫描到 1 ═══ 操作：push 1 栈状态： ┌───┐ │ 1 │ ← 栈顶 │ 3 │ │ 2 │ │ 5 │ └───┘ 栈 = [5, 2, 3, 1]

步骤5：扫描到 + → 计算！

═══ 步骤5：扫描到 + ═══ 操作：pop 1（右操作数），pop 3（左操作数） 3 + 1 = 4 push 4 计算前： 计算后： ┌───┐ ┌───┐ │ 1 │← 先弹出(右) │ │ │ 3 │← 后弹出(左) │ 4 │← 结果压入 │ 2 │ │ 2 │ │ 5 │ │ 5 │ └───┘ └───┘ 栈 = [5, 2, 3, 1] 栈 = [5, 2, 4]

步骤6：扫描到 * → 计算！

═══ 步骤6：扫描到 * ═══ 操作：pop 4（右），pop 2（左） 2 * 4 = 8 push 8 计算前： 计算后： ┌───┐ ┌───┐ │ 4 │← 先弹出(右) │ │ │ 2 │← 后弹出(左) │ 8 │← 结果压入 │ 5 │ │ 5 │ └───┘ └───┘ 栈 = [5, 2, 4] 栈 = [5, 8]

步骤7：扫描到 2 → push

═══ 步骤7：扫描到 2 ═══ 操作：push 2 栈状态： ┌───┐ │ 2 │ ← 栈顶 │ 8 │ │ 5 │ └───┘ 栈 = [5, 8, 2]

步骤8：扫描到 / → 计算！

═══ 步骤8：扫描到 / ═══ 操作：pop 2（右），pop 8（左） 8 / 2 = 4 push 4 计算前： 计算后： ┌───┐ ┌───┐ │ 2 │← 先弹出(右/除数) │ │ │ 8 │← 后弹出(左/被除数) │ 4 │← 结果压入 │ 5 │ │ 5 │ └───┘ └───┘ 栈 = [5, 8, 2] 栈 = [5, 4]
⚠️ 除法顺序：后缀表达式 8 2 / 意思是 8÷2=4（不是 2÷8）。
规则："先弹出的是右操作数（除数），后弹出的是左操作数（被除数）"。

步骤9：扫描到 + → 最后一步计算！

═══ 步骤9：扫描到 + ═══ 操作：pop 4（右），pop 5（左） 5 + 4 = 9 push 9 计算前： 计算后： ┌───┐ ┌───┐ │ 4 │← 先弹出(右) │ │ │ 5 │← 后弹出(左) │ 9 │← 最终结果！ └───┘ └───┘ 栈 = [5, 4] 栈 = [9] ← 表达式结束，栈中唯一元素 = 答案

四、等价中缀表达式还原

后缀 5 2 3 1 + * 2 / + 等价的中缀表达式为：

5 + ((2 * (3 + 1)) / 2)

逐步还原：

5 2 3 1 + * 2 / + 3 1 + → (3+1) = 4 2 4 * → 2*4 = 8 → 2*(3+1) 8 2 / → 8/2 = 4 → (2*(3+1))/2 5 4 + → 5+4 = 9 → 5 + (2*(3+1))/2 = 9

五、操作符运算规则总结

操作符弹出顺序计算示例
+ 先pop=右，后pop=左 左 + 右 3 1 + → 3+1=4
- 先pop=右，后pop=左 左 - 右 5 2 - → 5-2=3（不是2-5！）
* 先pop=右，后pop=左 左 * 右 2 4 * → 2*4=8
/ 先pop=右（除数），后pop=左（被除数） 左 / 右 8 2 / → 8/2=4（不是2/8！）
🎯 核心规则：后缀表达式中，操作符前面最近的两个操作数就是它的运算对象。先弹出的是右操作数，后弹出的是左操作数。减法和除法要特别注意顺序！

六、考试速算技巧

不用画栈，快速手算：从左到右扫描，找到第一个"操作数 操作数 操作符"的模式，立即计算并替换，重复直到只剩一个数字。

原始： 5 2 3 1 + * 2 / + └─┬─┘ 找到 3 1 + → 4，替换： → 5 2 4 * 2 / + └─┬─┘ 找到 2 4 * → 8，替换： → 5 8 2 / + └─┬─┘ 找到 8 2 / → 4，替换： → 5 4 + └─┬─┘ 找到 5 4 + → 9 ✅
📖 后缀表达式求值是第3章栈的经典应用，掌握"数字压栈、操作符弹两个算一个压回去"即可。
