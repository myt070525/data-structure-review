🍃 完全二叉树 — 叶子结点数推导

一棵完全二叉树有 321 个结点，
问叶子结点有多少个？
正确答案
161 个叶子结点
公式：n₀ = (n + 1) / 2 = (321 + 1) / 2 = 161

一、完全二叉树的定义与特征

完全二叉树（Complete Binary Tree）是一棵特殊的二叉树：

  • 除了最后一层可能不满外，其他每一层都是满的
  • 最后一层的结点都靠左排列，不会有"空位"在左边而右边有结点的情况

关键性质：完全二叉树中，度为1的结点最多只有1个（且只可能出现在倒数第二层最右边）。

完全二叉树示例（n=10）： ① ← 第1层（满） / \ ② ③ ← 第2层（满） / \ / \ ④ ⑤ ⑥ ⑦ ← 第3层（满） / \ / ⑧ ⑨ ⑩ ← 第4层（最后一层，靠左） 叶子结点（无孩子）：④,⑤,⑥,⑦,⑧,⑨,⑩？等等... 实际上：⑧,⑨,⑩,⑤,⑥,⑦ 是叶子（共6个） 注意④有孩子⑧⑨，不是叶子！

二、通用二叉树性质回顾

符号含义关系
n₀ 度为0的结点（叶子） n = n₀ + n₁ + n₂
n₁度为1的结点
n₂度为2的结点
二叉树通用性质 n₀ = n₂ + 1

性质 n₀ = n₂ + 1 对任何二叉树都成立（叶子数 = 度为2的结点数 + 1）。为什么？因为每个度为2的结点贡献2条边（指向孩子），每个度为1的结点贡献1条边，n个结点的树有n-1条边。

三、核心推导 — 完全二叉树叶子结点公式

对于完全二叉树，n₁ 只能是 0 或 1（最多一个度为1的结点）。

①
n = n₀ + n₁ + n₂
总结点 = 叶子 + 度1结点 + 度2结点
②
n₀ = n₂ + 1，即 n₂ = n₀ - 1
二叉树通用性质
③
代入：n = n₀ + n₁ + (n₀ - 1) = 2n₀ + n₁ - 1
④
移项：2n₀ = n - n₁ + 1
⑤
关键：n₁ 是 0 还是 1？

n₁ 的判定：

对于完全二叉树，n₁ 与总结点数 n 的奇偶性有关：

  • 如果 n 是偶数 → n₁ = 1（最后一个非叶子结点只有一个左孩子）
  • 如果 n 是奇数 → n₁ = 0（所有非叶子结点都有两个孩子）

本题 n = 321，是奇数 → n₁ = 0。

⑥
n₀ = (n - n₁ + 1) / 2 = (321 - 0 + 1) / 2 = 322 / 2 = 161

四、为什么 n₁ 由奇偶性决定？— 直观理解

完全二叉树按层编号（从1开始），结点 i 的孩子是 2i 和 2i+1。

n = 321（奇数）的情况： 编号1到321。最后一个有孩子的结点是 ⌊321/2⌋ = 160。 结点160的左孩子是 2×160=320，右孩子是 2×160+1=321。 320 ≤ 321 ✅ → 有左孩子 321 ≤ 321 ✅ → 有右孩子 → 结点160有两个孩子！→ 所有非叶子结点都有两个孩子 → n₁ = 0 n = 320（偶数）的情况（对比）： 最后一个有孩子的结点是 ⌊320/2⌋ = 160。 左孩子 320 ≤ 320 ✅ 右孩子 321 ≤ 320 ❌ → 没有右孩子！ → 结点160只有一个孩子！→ n₁ = 1

五、图解 321 个结点的完全二叉树

n = 321，n₀ = 161，n₁ = 0，n₂ = 160 树的结构（示意）： ① ← 第1层：1个结点 / \ ② ③ ← 第2层：2个结点 / \ / \ ④ ⑤ ⑥ ⑦ ← 第3层：4个结点 / \ / \ / \ / \ ... ... ... ... ... ← 中间层全满 / \ / \ / \ / \ ⑨⓪ ⑨① ⑨② ⑨③ ⑨④ ⑨⑤ ... ⑮⑨ ⑯⓪ ← 第8层：128个结点（全满） / \ / \ / \ / \ ... / \ ②⑤⑥②⑤⑦ ... ← 第9层：最后一层（部分） ... ... ③②⓪ ③②① ← 最后两个结点（320和321） 各层统计（满二叉树到第8层）： 第1层：2⁰ = 1 第2层：2¹ = 2 ... 第8层：2⁷ = 128 前8层总计 = 2⁸ - 1 = 255 第9层结点数 = 321 - 255 = 66 叶子结点 = 第9层的全部66个 + 第8层中没有孩子的结点 第8层结点共128个，有孩子的结点数 = ⌈66/2⌉ = 33 第8层叶子 = 128 - 33 = 95 总叶子 = 66 + 95 = 161 ✅

六、完整公式总结

n₀ = ⌈n / 2⌉（向上取整）

七、记忆口诀与易错点

记
"完全二叉树叶子数 = 总结点数 ÷ 2 向上取整"
或者记："奇数加1除以2，偶数直接除以2"
⚠️ 常见错误：
• 把完全二叉树和满二叉树混淆。满二叉树每层都满，是完全二叉树的特例。
• 忘记 n₁ 只能是 0 或 1！如果有2个以上度为1的结点，就不是完全二叉树了。
• 公式 n₀ = n₂ + 1 对所有二叉树都成立，但 n₀ = (n+1)/2 只对 n₁=0 的完全二叉树成立！
🎯 同类变式：
• 已知叶子数 n₀，求总结点数 n → n = 2n₀ - 1（当 n₁=0）
• 已知总结点数 n，求度为2的结点 n₂ → n₂ = (n-1)/2（当 n₁=0）
• 非完全二叉树 → 不能用简化公式，必须逐层分析！
📖 完全二叉树的叶子结点公式是第6章的高频考点。记住：n为奇数→n₀=(n+1)/2，n为偶数→n₀=n/2。
