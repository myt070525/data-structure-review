🧅 广义表表尾 — 剥洋葱式求解

广义表 L = ((a,b), (a,(c,d)))
则 Tail(L) 的结果是？
正确答案
Tail(L) = ((a,(c,d)))
即去掉第一个元素 (a,b)，剩下的元素组成新表

一、先看清广义表 L 的结构

广义表是线性表的推广，元素可以是原子（单个数据）或子表（又是一个表）。

L = ( (a,b) , (a,(c,d)) ) └─┬──┘ └───┬────┘ │ │ 元素1(Elem₁) 元素2(Elem₂) 是一个子表 也是一个子表 里面是a和b 里面是a和(c,d) 通俗地看： L 是一个有两样东西的"大盒子"： · 第1个格子里 → 一个小盒子装 a 和 b · 第2个格子里 → 一个小盒子装 a 和 另一个更小的盒子(c,d)

二、Head 与 Tail 的核心定义

📌 Head（表头）— 取第一个元素

取广义表的第一个元素，原样返回。如果第一个元素是子表，就返回那个子表（保留括号）。

// 例子 Head( ((a,b), (a,(c,d))) ) = (a,b) // 第一个元素是子表(a,b) Head( (a, b, c) ) = a // 第一个元素是原子a

📌 Tail（表尾）— 去掉第一个，剩下的包一层括号

去掉第一个元素后，剩余所有元素组成一个新表。Tail 永远返回一个表（有外层括号）！

// 例子 Tail( ((a,b), (a,(c,d))) ) = ((a,(c,d))) // 去掉(a,b)，剩下(a,(c,d))，包成表 Tail( (a, b, c) ) = (b, c) // 去掉a，b和c组成表(b,c) Tail( (a) ) = () // 去掉a，没剩东西 → 空表
⚠️ 最容易错的地方：Tail 多一层括号！
Tail((a,b)) = (b)，不是 b。
因为 Tail 返回的是"剩余元素组成的表"，即使只剩一个元素也要包括号。

三、剥洋葱 —— 逐层可视化

本题：Tail(L)，只有一步

原
L = ( (a,b) , (a,(c,d)) )
砍
砍掉第一个元素 (a,b)，剩余 (a,(c,d))
包
剩余元素组成新表 → ( (a,(c,d)) ) ← 注意这层外层括号！
✅ Tail(L) = ((a,(c,d)))

洋葱透视：看清每一层括号的作用

═══ 逐层剥洋葱：Tail(L) ═══ 原始表 L： ┌─────────────────────────────────────┐ │ ( (a,b) , (a,(c,d)) ) │ │ ↑ ↑ │ │ 元素1 元素2 │ │ 第1块洋葱 第2块洋葱 │ └─────────────────────────────────────┘ Tail 操作 = 剥掉第1块洋葱皮（元素1），剩下的包起来： ┌──────────────────────┐ │ ( (a,(c,d)) ) │ ← 新表 │ └──┬──────┘ │ │ │ │ │ │ │ 唯一的元素 外层括号│ │ (一个子表) (Tail加的)│ └──────────────────────┘ 最终结果：((a,(c,d))) · 最外层括号 → Tail 加的（因为它返回一个"表"） · 内层括号 → 元素2本身就是一个子表

四、深入探究 L 的内部结构

元素1：(a,b)

(a,b) ├── a (原子) └── b (原子)

Head((a,b)) = a；Tail((a,b)) = (b)

元素2：(a, (c,d))

(a, (c,d)) ├── a (原子) └── (c,d) (子表) ├── c (原子) └── d (原子)

Head((a,(c,d))) = a；Tail((a,(c,d))) = ((c,d))

Tail(L) 的结果 ((a,(c,d))) 进一步操作：

Head(Tail(L)) = Head(((a,(c,d)))) = (a,(c,d))

Tail(Tail(L)) = Tail(((a,(c,d)))) = ()（空表！因为只有一个元素，去掉就空了）

五、完整对比表

表达式结果说明
Head(L)(a,b)取第一个元素
Tail(L)((a,(c,d)))去掉第一个，剩余包括号 ← 本题答案
Head(Head(L))a对(a,b)取Head → a
Tail(Head(L))(b)对(a,b)取Tail → (b)
Head(Tail(L))(a,(c,d))对((a,(c,d)))取Head
Tail(Tail(L))()去掉唯一元素 → 空表
Head(Head(Tail(L)))a深入元素2的第一个
Tail(Head(Tail(L)))((c,d))深入元素2的剩余

六、防踩坑指南

坑位错误做法正确做法
坑① Tail((a,b))写成b 正确是(b)，Tail永远加括号
坑② Head((a))写成(a) 正确是a，Head不加额外括号
坑③ 忘了数括号层数 每做一步，写下结果，数清楚括号匹配
坑④ Tail(L)写成(a,(c,d)) 少了一层外层括号！应是((a,(c,d)))
🧠 记忆口诀：
• Head = "取第一个，原样返还"（不加不减括号）
• Tail = "砍第一个，剩的包一层"（永远多一层外层括号）
• 嵌套表达式 = "从内往外剥，一次做一步"

七、同类题自测

以本题为基础，请计算以下表达式：

  1. Head(L) = (a,b)
  2. Tail(L) = ((a,(c,d))) ← 本题
  3. Head(Tail(L)) = (a,(c,d))
  4. Tail(Tail(L)) = ()
📖 广义表的 Head/Tail 是第5章的基础操作，关键记住：Head不加括号，Tail加一层括号。
