测验2026春 · 选择填空 · 第8题

🌳 二叉树左子树 — 中序+后序重建

二叉树的中序遍历序列：A B C D E F G
后序遍历序列：B D C A F G E
问：该二叉树的左子树有几个结点？
正确答案
左子树有 4 个结点
根结点为 E（后序最后一个），左子树包含 A, B, C, D

一、三种遍历的核心特征

遍历方式顺序关键特征
先序 (Preorder) 根 → 左 → 右 第一个是根
中序 (Inorder) 左 → 根 → 右 根把序列分为左右子树
后序 (Postorder) 左 → 右 → 根 最后一个是根

本题给了中序+后序，这是最常见的"已知两种遍历，重建二叉树"题型。

二、重建步骤 — 从后序找根，从中序分左右

核心算法（递归）：

1
后序序列的最后一个结点 = 整棵树的根。本题后序最后是 E → 根是 E。
2
在中序序列中找到根 E。中序 = A B C D E F G。E 左边的 {A,B,C,D} 是左子树，右边的 {F,G} 是右子树。
3
递归重建左子树和右子树。从后序中按长度切分出左子树的后序和右子树的后序。

三、完整重建过程 — ASCII图解

第1层：确定整棵树的根

中序：A B C D E F G 后序：B D C A F G E ← 最后一个是根！ 中序中 E 左边 = {A,B,C,D} → 左子树（4个结点） 中序中 E 右边 = {F,G} → 右子树（2个结点） 后序中左子树部分 = 前4个：{B,D,C,A} 后序中右子树部分 = 接着2个：{F,G} 后序中根 = 最后一个：{E} E / \ {A,B,C,D} {F,G} (左子树) (右子树) 4个结点 2个结点

第2层：重建左子树（中序=A,B,C,D，后序=B,D,C,A）

左子树中序：A B C D 左子树后序：B D C A ← 最后一个A是左子树的根！ 中序中 A 左边 = {} → 左子树的左子树为空 中序中 A 右边 = {B,C,D} → 左子树的右子树（3个结点） E / \ A {F,G} \ {B,C,D}

第3层：重建{A,B,C,D}中A的右子树（中序=B,C,D，后序=B,D,C）

右子树(A的)中序：B C D 右子树(A的)后序：B D C ← 最后一个C是根！ 中序中 C 左边 = {B} → 左子树（1个结点） 中序中 C 右边 = {D} → 右子树（1个结点） E / \ A {F,G} \ C / \ B D

第4层：重建右子树{F,G}（中序=F,G，后序=F,G,?）

右子树(E的)中序：F G 右子树(E的)后序：F G ← 最后一个G是根！ 中序中 G 左边 = {F} → 左子树（1个结点） 中序中 G 右边 = {} → 右子树为空 E / \ A G \ / C F / \ B D

四、完整二叉树结构

E / \ / \ A G \ / \ / C F / \ / \ B D 左子树（E的左孩子A及其后代）：A, C, B, D → 共4个结点 右子树（E的右孩子G及其后代）：G, F → 共2个结点 根：E → 1个结点 总计：7个结点

五、验证：遍历一致性检查

用ASCII树验证三种遍历结果是否与题目一致：

树结构： E / \ A G \ / C F / \ B D 先序（根左右）：E → A → C → B → D → G → F = E A C B D G F 中序（左根右）：A → B → C → D → E → F → G = A B C D E F G ✅ 与题目一致！ 后序（左右根）：B → D → C → A → F → G → E = B D C A F G E ✅ 与题目一致！

✅ 验证通过！重建正确。

六、答题总结

问题：左子树有几个结点？

从重建的树可知，E的左子树包含：A（直接左孩子）、C、B、D。共4个结点。

🎯 快速解法（不用完整重建）：
① 后序最后 = 根 E
② 中序中 E 左边的元素个数 = 左子树结点数
③ 中序 = A B C D E F G → E 左边 4 个 → 答案：4
只需两步！不需要完整画出树。
⚠️ 注意：题目问的是"左子树有几个结点"，不是"左孩子有几个"。左子树包含左孩子及其所有后代。如果只数直接左孩子（A一个），那就错了！

七、同类题型技巧

已知条件能唯一确定二叉树？关键步骤
先序 + 中序 ✅ 能 先序第一个=根，中序划分左右
中序 + 后序 ✅ 能（本题） 后序最后一个=根，中序划分左右
先序 + 后序 ❌ 不一定 无法唯一确定（除非是满二叉树等特殊情况）
只有一种遍历 ❌ 不能 信息不足
📖 根据遍历序列重建二叉树是第6章的核心技能。记住：后序最后=根，中序划分左右子树。
