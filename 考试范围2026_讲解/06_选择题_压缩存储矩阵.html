📐 矩阵的压缩存储

下列矩阵中，可以进行压缩存储的是（　）。

A. 对称矩阵    B. 三角矩阵    C. 对角矩阵    D. 以上都可以
正确答案
D — 以上都可以
对称矩阵、三角矩阵、对角矩阵都能压缩存储

一、为什么要压缩存储？

很多特殊矩阵有大量重复元素或零元素。如果按普通二维数组存储，会浪费大量空间。

压缩存储的核心思想：只存储有用的（非零/不重复的）元素，通过公式计算位置。

矩阵类型n×n矩阵的元素总数压缩后存储量节省比例
普通矩阵n²n²0%
对称矩阵n²n(n+1)/2≈ 50%
三角矩阵n²n(n+1)/2 + 1≈ 50%
三对角矩阵n²3n-2≈ 1 - 3/n

二、对称矩阵 —— 以对角线为轴，两边对称

定义：aij = aji，即关于主对角线对称。存储时可以只存下三角（或上三角）。

对称矩阵（4×4） 列: 0 1 2 3 行 ┌────┬────┬────┬────┐ 0 │ 1 │ 2 │ 3 │ 4 │ ├────┼────┼────┼────┤ 1 │ 2 │ 5 │ 6 │ 7 │ ← 注意：a[0][1]=2, a[1][0]=2 (相等!) ├────┼────┼────┼────┤ 2 │ 3 │ 6 │ 8 │ 9 │ ├────┼────┼────┼────┤ 3 │ 4 │ 7 │ 9 │ 10 │ └────┴────┴────┴────┘ 压缩存储（只存下三角，按行优先） 一维数组 B[k]：[1, 2, 5, 3, 6, 8, 4, 7, 9, 10] 元素个数 = n(n+1)/2 = 4×5/2 = 10个（原始16个→节省37.5%） 地址映射公式（下三角，行优先，i≥j）： k = i(i+1)/2 + j （0-based下标）
// 访问对称矩阵元素 a[i][j] int getSym(int B[], int i, int j) { if (i >= j) return B[i*(i+1)/2 + j]; // 下三角 else return B[j*(j+1)/2 + i]; // 利用对称性，转置访问 }

三、三角矩阵 —— 对角线一侧全是同一个常数

定义：对角线以上（或以下）全是同一个常数（通常是0）。上三角矩阵：下三角全为常数；下三角矩阵：上三角全为常数。

下三角矩阵（4×4），上三角全为0 列: 0 1 2 3 行 ┌────┬────┬────┬────┐ 0 │ 1 │ 0 │ 0 │ 0 │ ├────┼────┼────┼────┤ 1 │ 2 │ 5 │ 0 │ 0 │ ├────┼────┼────┼────┤ 2 │ 3 │ 6 │ 8 │ 0 │ ├────┼────┼────┼────┤ 3 │ 4 │ 7 │ 9 │ 10 │ └────┴────┴────┴────┘ 压缩存储：下三角 n(n+1)/2 个元素 + 1个常数 一维数组 B[k]：[1, 2, 5, 3, 6, 8, 4, 7, 9, 10, 0] 元素个数 = n(n+1)/2 + 1 = 11个

四、对角矩阵 —— 只有对角线附近有非零元素

定义：所有非零元素集中在主对角线及其两侧若干条对角线上。最常见的是三对角矩阵。

三对角矩阵（5×5）—— 只有3条对角线有非零元素 列: 0 1 2 3 4 行 ┌────┬────┬────┬────┬────┐ 0 │ a │ b │ 0 │ 0 │ 0 │ ├────┼────┼────┼────┼────┤ 1 │ c │ d │ e │ 0 │ 0 │ ├────┼────┼────┼────┼────┤ 2 │ 0 │ f │ g │ h │ 0 │ ├────┼────┼────┼────┼────┤ 3 │ 0 │ 0 │ i │ j │ k │ ├────┼────┼────┼────┼────┤ 4 │ 0 │ 0 │ 0 │ l │ m │ └────┴────┴────┴────┴────┘ 压缩存储（按行优先，只存3条对角线） 一维数组 B[k]：[a,b, c,d,e, f,g,h, i,j,k, l,m] 元素个数 = 3n - 2 = 3×5 - 2 = 13个（原始25个→节省48%） 地址映射公式（三对角，行优先，|i-j|≤1）： k = 2i + j （简化公式，第0行特殊处理）

五、三种压缩矩阵对比汇总

矩阵类型特点压缩后元素数压缩条件
对称矩阵 aij = aji n(n+1)/2 只存上三角或下三角
三角矩阵 上/下三角全为常数c n(n+1)/2 + 1 存非三角部分+常数c
对角矩阵 仅对角线附近非零 3n-2（三对角） 只存对角线带
🔑 共同点：这三种矩阵都有冗余信息（重复值或零值），因此都可以压缩存储。压缩的本质是用空间换时间的反向操作——用计算时间换存储空间。

六、逐项排查

选项矩阵类型可压缩？压缩方式
A 对称矩阵 ✅ 可以 只存上三角或下三角
B 三角矩阵 ✅ 可以 存非三角区+常数
C 对角矩阵 ✅ 可以 只存对角线带
D 以上都可以 ✅ 全部可以！ 三种都是经典压缩存储场景

七、地址映射公式速查

矩阵类型存储方式下标k = ?（0-based，行优先）
对称矩阵（下三角） i ≥ j 时存，否则用aji k = i(i+1)/2 + j
下三角矩阵 i ≥ j 存具体值；i < j 存常数c k = i(i+1)/2 + j（i≥j）
最后位置存c
三对角矩阵 |i-j| ≤ 1 存值，否则不存 k = 2i + j（i=0时k=j）
🎯 考试重点：对称矩阵的下标计算公式 k = i(i+1)/2 + j 是高频考点，务必记住！推导思路：第 i 行前面有 i 行（0到i-1），共 i(i+1)/2 个元素，加上本行偏移 j。
