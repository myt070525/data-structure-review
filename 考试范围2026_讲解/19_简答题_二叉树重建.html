📐 简答第1题 · 二叉树重建

先序 BCDEAFG + 中序 DECFAGB → 后序 EDFGACB

后序遍历序列：E D F G A C B
根 = B，递归分割左右子树

一、重建原理

已知先序遍历和中序遍历可以唯一确定一棵二叉树。核心原理：

  • 先序序列的第一个元素一定是根结点
  • 在中序序列中找到根的位置 → 左边是左子树，右边是右子树
  • 递归地对左右子树重复上述过程

二、ASCII分步重建二叉树

1 确定根结点

先序: B C D E A F G → 根 = B 中序: D E C F A G B ↑ 等一下，让我仔细看： 先序 = B C D E A F G（7个字符） 中序 = D E C F A G B（题目说 DECFAGB） 中序中 B 在最后： 中序: D E C F A G B ├─ 左子树(6个) ─┤ 根 所以 B 的左子树包含 {D,E,C,F,A,G}，右子树为空。

2 重建B的左子树（6个结点）

左子树的先序（从先序中取B之后的6个）：C D E A F G 左子树的中序（从中序中B前面的6个）： D E C F A G 先序第一个 = C → C是左子树的根 中序中C的位置： 中序: D E C F A G ├左─┤ ↑ ├─右─┤ C的左子树中序 = {D, E} (2个结点) C的右子树中序 = {F, A, G} (3个结点)

3 递归构建C的左子树

左子树先序（从先序中取C后的2个）：D E 左子树中序： D E 先序第一个 = D → D是根 中序: D E → D的左子树为空，右子树={E} D的右子树： 先序 = E，中序 = E → E是叶子结点 C / D \ E

4 递归构建C的右子树

右子树先序（取D,E之后）：A F G 右子树中序： F A G 先序第一个 = A → A是根 中序: F A G → A的左子树={F}，右子树={G} F是单结点（叶子），G是单结点（叶子） C / \ D A \ / \ E F G

5 完整的二叉树

B / C / \ D A \ / \ E F G 验证先序: B → C → D → E → A → F → G = BCDEAFG ✓ 验证中序: D → E → C → F → A → G → B = DECFAGB ✓

三、求后序遍历序列

后序遍历规则：左 → 右 → 根

(D的左子树: 空 → 跳过) (D的右子树: E) 遍历E: E没有孩子 → 输出 E (回到D) → 输出 D (A的左子树: F) → 输出 F (A的右子树: G) → 输出 G (回到A) → 输出 A (回到C) → 输出 C (B的右子树: 空) (回到B) → 输出 B ───────────────────────────── 后序序列: E D F G A C B
后序 = E D F G A C B ✅

四、重建算法的通用步骤

1 从先序序列中取第一个元素作为当前子树的根
2 在中序序列中找到该根的位置，将中序分为左、右两部分
3 根据左子树中序的长度，从先序中切出左子树先序和右子树先序
4 递归地对左右子树重复步骤1-3
5 当序列为空时，递归结束（空树）
⚠️ 注意：仅知先序和后序不能唯一确定二叉树。必须知道中序才能唯一确定。因为中序区分了左右子树。
