🌳 填空第4题 · 哈夫曼树

一棵哈夫曼树有125个结点，叶结点有几个？

63
个叶结点

一、哈夫曼树的特殊性质

哈夫曼树是通过贪心合并构建的最优二叉树。它的关键特征：哈夫曼树只有度为0（叶子）和度为2（内部）的结点，没有度为1的结点。

为什么？因为每次合并都是把两棵子树合到一个新根下，新根一定有两个孩子→度=2。

二、推导过程

①
n = n₀ + n₂
总结点 = 叶结点 + 内部结点（没有度为1的结点）
②
n₀ = n₂ + 1
二叉树通用性质：叶子数 = 度为2的结点数 + 1
③
把②代入①：n = n₀ + (n₀ - 1) = 2n₀ - 1
④
n₀ = (n + 1) / 2 = (125 + 1) / 2 = 63
🎯 万能公式：对哈夫曼树（无度为1结点）→ 叶子数 = (总结点数 + 1) / 2。反向：总结点 = 2×叶子数 - 1。永远记住这个！
💡 通俗类比：哈夫曼树像一场淘汰赛。每场比赛（内部结点）淘汰1个人，最后冠军（根）以上没人了。N个选手参赛，需要N-1场比赛。叶子=选手，内部结点=比赛。所以叶子比内部结点多1。
