广义表的 Head 与 Tail 运算

广义表 A = ((a,b), (c,(d),((e,f),g)))
则 Head(Tail(Head(Tail(A)))) 的值为？
正确答案
D — (d)

一、先搞懂：广义表是什么？

广义表（Generalized List）是线性表的推广。线性表的每个元素只能是单个数据，而广义表的元素可以是：

  • 原子（单个数据，如 a、b、3）
  • 子表（又是一个广义表，如 (a,b)）
A = ( (a,b) , (c,(d),((e,f),g)) ) ↑ ↑ 第1个元素(子表) 第2个元素(子表) 更直观地看： A = ( 元素₁ , 元素₂ ) = ( (a,b) , (c,(d),((e,f),g)) )

这个广义表 A 里面有 2 个元素，每个元素本身又是一个子表。

二、核心定义：Head 和 Tail 到底怎么算？

📌 Head（表头）

取广义表的第一个元素。注意：返回的是元素本身，不额外加括号。

Head( (a,b) ) = a // 第一个元素是原子 a Head( ( (a,b), c ) ) = (a,b) // 第一个元素是子表 (a,b)，原样返回

📌 Tail（表尾）

去掉第一个元素后，剩余元素组成的新表。注意：Tail 永远返回一个表（有外层括号）。

Tail( (a,b) ) = (b) // 去掉 a，剩下 b，组成表 (b) ← 注意有括号！ Tail( (a) ) = ( ) // 去掉 a，没剩东西 → 空表 () Tail( (a,b,c) ) = (b,c) // 去掉 a，剩下 b,c 组成表 (b,c)
⚠️ 最容易错的地方：
Tail( (a,b) ) 的结果是 (b)，不是 b。
Tail 永远在结果外面包一层括号，因为它返回的是"剩余元素组成的表"。
即使只剩一个元素，也要变成 (b) 而不是 b。

三、一步一步推导完整表达式

要计算 Head(Tail(Head(Tail(A))))，从最内层开始，像剥洋葱一样：

步骤操作详细解释
第①步 Tail(A) A = ((a,b), (c,(d),((e,f),g)))
A 有两个元素：① (a,b)　② (c,(d),((e,f),g))
去掉第①个元素 (a,b)，剩下的组成表：
→ ( (c,(d),((e,f),g)) )
注意：结果被包了一层括号，只有一个元素 (c,(d),((e,f),g))
第②步 Head( Tail(A) ) 对 ((c,(d),((e,f),g))) 取 Head（第一个元素）：
→ (c, (d), ((e,f),g))
这是一个有 3 个元素的子表：c、(d) 和 ((e,f),g)
第③步 Tail( Head(Tail(A)) ) 对 (c, (d), ((e,f),g)) 取 Tail（去掉第一个元素 c）：
剩下 (d) 和 ((e,f),g) 两个元素，组成新表：
→ ( (d), ((e,f),g) )
第④步 Head( Tail(Head(Tail(A))) ) 对 ( (d), ((e,f),g) ) 取 Head（第一个元素）：
→ (d) ✅

四、图解整个推导过程

A └── ( (a,b), (c,(d),((e,f),g)) ) ← 原表，2个元素 │ │ │ └── 元素2: (c,(d),((e,f),g)) └── 元素1: (a,b) ① Tail(A) = 去掉元素1，剩余元素组成的表 = ( (c,(d),((e,f),g)) ) ← 注意外层括号！只有1个元素的表 ② Head( Tail(A) ) = 取上面这个表的第一个元素 = (c, (d), ((e,f),g)) ← 这是一个3元素子表 │ │ │ │ │ └── 元素3: ((e,f),g) │ └── 元素2: (d) └── 元素1: c（原子） ③ Tail( Head(Tail(A)) ) = 去掉 c，剩下两个元素组成新表 = ( (d), ((e,f),g) ) ← 2个元素的表 ④ Head( ... ) = 取第一个元素 = (d) ← 🎯 答案！

五、常见踩坑点

❌ 错误1 把 Tail((a,b)) 当成了 b Tail 永远返回表 → 应该是 (b)
❌ 错误2 把 Head(( (a,b) )) 当成了 a Head 取第一个元素原样返回 → (a,b)
❌ 错误3 括号层层数数错了 每做一步，把结果写下来，数清楚括号匹配

六、记忆口诀

🧠 Head = 取第一个，原样返还
✂️ Tail = 砍第一个，剩下的包一层括号
🧅 嵌套表达式 = 从内往外剥，一次做一步
💡 自检方法：做完每一步，问自己两件事：
① 结果里有几个元素？（一个还是多个）
② 有没有正确加/去外层的括号？
全部正确后，再进入下一步。

七、自己动手验证

以本题为例，检验你是否真的懂了。请填空（先自己做，再看答案）：

  1. Head(A) = (a,b)
  2. Tail(A) = ((c,(d),((e,f),g)))
  3. Head(Head(Tail(A))) = c
  4. Tail(Tail(A)) = () —— 因为 Tail(A) 只有一个元素，去掉它就空了
📖 广义表是第5章的内容，Head/Tail 是广义表最基本的两个操作，务必熟练掌握。
